# FuelLocator

Android-приложение для мониторинга наличия и видов отпуска топлива на автозаправочных станциях (АЗС). Позволяет автолюбителям оперативно получать информацию о наличии бензина, дизеля и газа на заправках, прокладывать маршруты и вносить свой вклад в актуализацию данных.

---

## 🚀 Основные возможности

* **Интерактивная карта (Yandex MapKit)**:
  * **Кластеризация маркеров**: Заправки группируются в зависимости от масштаба карты. При клике на кластер происходит плавное приближение.
  * **Кастомные маркеры**: АЗС отображаются в виде колонок. Цвет указывает на лучшую доступность топлива (Зеленый — свободная продажа, Желтый — по картам/талонам, Красный — временно отсутствует, Серый — нет данных, Фиолетовый — добавлено пользователями).
  * **Яндекс-стиль геопозиции**: Отображение местоположения пользователя со стрелкой направления движения и радиусом погрешности.

* **Управление базой АЗС**:
  * **Добавление новых АЗС**: Добавление станции долгим нажатием на карту. Доступен удобный импорт координат из буфера обмена (ссылки Яндекс.Карт).
  * **Удаление АЗС**: Авторизованные пользователи могут удалить АЗС с карты, выбрав одну из причин (закрылась, дубликат, неверные координаты, другое). Запись об удалении логируется в облачную коллекцию `deletions` в Firestore.
  * **Редактирование топлива**: Возможность обновить статус каждого вида топлива (АИ-92, АИ-95, АИ-95+, АИ-100, ДТ, ДТ+, Газ) и указать вид отпуска (наличные, карты, талоны).

* **Службы и Уведомления**:
  * **Фоновый мониторинг**: Интеграция с Android `WorkManager` для периодической фоновой проверки изменений цен/наличия на АЗС с выводом системных push-уведомлений в реальном времени.

## 🛠️ Стек технологий

* **Язык**: Kotlin
* **UI**: Jetpack Compose & Material Design 3
* **Локальная БД**: Room Database (SQLite)
* **Облачная БД и Auth**: Firebase Firestore & Firebase Authentication (Email/Пароль и вход по номеру телефона)
* **Картография**: Yandex MapKit SDK v4
* **Внедрение зависимостей**: Dagger Hilt
* **Фоновые задачи**: WorkManager
* **Архитектура**: MVVM / Clean Architecture

## ⚙️ Установка и запуск

Для сборки проекта на вашем компьютере выполните следующие шаги:

### 1. Клонирование репозитория
```bash
git clone https://github.com/Kastyel1990/FuelLocator.git
cd FuelLocator
```

### 2. Настройка API Yandex MapKit
В корне проекта в файле `local.properties` (создайте его, если нет) укажите ваш API-ключ от Яндекс Карт:
```properties
MAPKIT_API_KEY=ваш_api_ключ_yandex_mapkit
```

### 3. Подключение Firebase
Поместите файл конфигурации `google-services.json`, полученный из вашей консоли Firebase, в папку `app/`:
```
FuelLocator/
  └── app/
       └── google-services.json
```

### 4. Сборка проекта
Для компиляции debug-версии выполните команду:
```bash
# Для Windows:
$env:JAVA_HOME="C:\Program Files\Android\Android Studio\jbr"
.\gradlew.bat assembleDebug

# Для Linux/macOS:
./gradlew assembleDebug
```
Готовый APK-файл будет находиться по пути: `app/build/outputs/apk/debug/app-debug.apk`.
